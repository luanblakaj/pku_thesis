# pku_thesis

This repository includes the analysis code for the thesis.

What is included
- `scripts/`: main analysis workflows.
- `R/`: helper functions and shared code.
- `thesis_france_part.Rproj`: open this in RStudio.
- `data/`, `docs/`, `outputs/`: empty placeholder folders only.

How to use
1. Open `thesis_france_part.Rproj` in RStudio.
2. Put your input data inside `data/` locally.
3. Run `scripts/00_run_all.R` or the specific script you need.

Notes
- No data, docs, or outputs are stored in this repository.
- Keep your data files local and do not commit them.
